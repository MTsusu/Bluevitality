#### 测试环境 Tomcat8 以下的变量定义添加在96行之下，具体百度
```
JAVA_OPTS="
$JAVA_OPTS
-server 
-Xms256M
-Xmx512M
-Xss1M
-Djava.awt.headless=true 
-Dfile.encoding=UTF-8
-Duser.country=CN
-Duser.timezone=Asia/Shanghai
-XX:MinHeapFreeRatio=80 
-XX:MaxHeapFreeRatio=80 
-XX:ThreadStackSize=512
-XX:NewSize=256m
-XX:NewRatio=4 
-XX:SurvivorRatio=8
-XX:+AggressiveOpts"
```
